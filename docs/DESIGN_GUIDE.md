# デザインガイドライン

このドキュメントでは、インフォグラフィックやトップ画像のデザインを統一するためのガイドラインを定義します。

---

## 🎨 ブランドカラーパレット

### メインカラー
- **プライマリーブルー**: `#3B82F6`
  - 用途: 見出し、重要な要素、CTA
  - RGB: (59, 130, 246)

### アクセントカラー
- **オレンジ**: `#F59E0B`
  - 用途: 強調、アクション、ポイント
  - RGB: (245, 158, 11)

- **グリーン**: `#10B981`
  - 用途: 成功、ポジティブな情報
  - RGB: (16, 185, 129)

### ベースカラー
- **ライトグレー(背景)**: `#F8FAFC`
  - 用途: 背景、カード背景
  - RGB: (248, 250, 252)

- **ダークグレー(テキスト)**: `#1E293B`
  - 用途: 本文テキスト
  - RGB: (30, 41, 59)

- **ミディアムグレー**: `#64748B`
  - 用途: 補足テキスト、キャプション
  - RGB: (100, 116, 139)

### カラーコード一覧(コピー用)
```
プライマリーブルー: #3B82F6
オレンジ: #F59E0B
グリーン: #10B981
ライトグレー: #F8FAFC
ダークグレー: #1E293B
ミディアムグレー: #64748B
```

#### インフォグラフィック(1200x630px)

- **キャプション**: 14-16px

#### トップ画像(1280x670px)
- **メインタイトル**: 56-72px

- **本文**: 20-28px
- **キャプション**: 16-18px

## 🖼️ レイアウト原則

### グリッドシステム
- **マージン**: 上下左右 60px
- **カラム**: 12カラムグリッド
- **ガター**: 20px

### 余白(スペーシング)
- **セクション間**: 40-60px
- **要素間**: 20-30px
- **テキスト行間**: 1.5-1.8倍

### 配置原則
1. **視線の流れ**: 左上→右上→左下→右下(Z型)
2. **重要度の配置**: 中央または左上に配置
3. **バランス**: 左右対称または視覚的バランスを保つ

## 📐 インフォグラフィックテンプレート

### テンプレート1: シンプル型

```
┌─────────────────────────────────────┐
│  [ロゴ/アイコン]                    │
│                                     │
│         [メインタイトル]            │
│         [サブタイトル]              │
│                                     │
│  ┌─────────┐ ┌─────────┐          │
│  │ポイント1│ │ポイント2│          │
│  └─────────┘ └─────────┘          │
│                                     │
│  [補足情報・出典]                   │
└─────────────────────────────────────┘
```

**用途**: 2-3個のポイントを強調したい場合

### テンプレート2: リスト型

```
┌─────────────────────────────────────┐
│  [メインタイトル]                   │
│                                     │
│  ① [ポイント1]                      │
│     [説明文]                        │
│                                     │
│  ② [ポイント2]                      │
│     [説明文]                        │
│                                     │
│  ③ [ポイント3]                      │
│     [説明文]                        │
│                                     │
│  [ロゴ/出典]                        │
└─────────────────────────────────────┘
```

**用途**: 手順やステップを説明したい場合

### テンプレート3: 比較型

```
┌─────────────────────────────────────┐
│         [メインタイトル]            │
│                                     │
│  ┌───────────┐  VS  ┌───────────┐  │
│  │  Before   │      │   After   │  │
│  │           │      │           │  │
│  │ [内容]    │      │ [内容]    │  │
│  └───────────┘      └───────────┘  │
│                                     │
│  [結論・メッセージ]                 │
└─────────────────────────────────────┘
```

**用途**: Before/Afterや比較を示したい場合

## 🎨 CANVA用設定

### トップ画像テンプレート作成手順

1. **新規デザイン作成**
   - サイズ: カスタム 1280x670px

2. **背景設定**
   - 背景色: #F8FAFC
   - またはグラデーション: #F8FAFC → #E2E8F0

3. **要素配置**
   - 左上: ロゴまたはアイコン(120x120px)
   - 中央: メインタイトル(Noto Sans JP Bold, 56-72px)
   - 下部: サブタイトルまたはキャッチコピー(32-40px)

4. **装飾**
   - アクセントカラーのライン(4-8px)
   - シンプルな図形(円、四角)
   - アイコン(Heroicons風)

5. **テンプレート保存**
   - 「テンプレート」として保存
   - 毎回このテンプレートから作成
