## 3. 記事作成用プロンプト

### 基本プロンプト

```
以下の調査結果をもとに、Note記事を作成してください。

【記事ID】: [XXX]
【タイトル】: [記事タイトル]

【調査結果】:
[research-XXX.mdの内容を貼り付け]

【記事の条件】:
- **ターゲット読者**: 副業を始めたい・効率化したい会社員
- **読了時間**: 5-7分(2000-3000文字)
- **トーン**: 親しみやすく、実践的で、専門性を感じさせる
- **構成**: イントロ、3つのメインセクション、まとめ

【記事構成】:
1. **イントロダクション**(200-300文字)
   - 読者の課題を明確にする
   - この記事で得られることを提示

2. **セクション1**: [見出し]
   - 具体的な内容
   - 実例やデータを含める

3. **セクション2**: [見出し]
   - 具体的な内容
   - 実例やデータを含める

4. **セクション3**: [見出し]
   - 具体的な内容
   - 実例やデータを含める

5. **まとめ**(200-300文字)
   - 重要なポイントを3つにまとめる
   - 次のアクションを提示

【出力形式】:
以下のMarkdown形式で出力してください:

# [記事タイトル]

## イントロダクション
[本文]

## [セクション1の見出し]
[本文]

**ポイント**:
- 
- 

## [セクション2の見出し]
[本文]

**具体例**:
[例]

## [セクション3の見出し]
[本文]

## まとめ
[本文]

**重要なポイント**:
1. 
2. 
3. 

---

【メタ情報も生成してください】:
- **キーワード**: 5個
- **ハッシュタグ**: 5個
- **推定読了時間**: X分
```

## 4. LinkedIn向けリライトプロンプト

### 基本プロンプト

```
以下のNote記事を、LinkedIn向けにリライトしてください。

【元記事】:
[Note記事の本文を貼り付け]

【リライトの条件】:
- **文字数**: 1000-1500文字(Note記事の50-70%)
- **トーン**: よりビジネス向け、プロフェッショナル
- **構成**: 短い段落、読みやすさ重視
- **絵文字**: 適度に使用(3-5個程度)
- **ハッシュタグ**: 3-5個

【出力形式】:
以下の形式で出力してください:

---
[リライトした本文]

#ハッシュタグ1 #ハッシュタグ2 #ハッシュタグ3
---

【投稿用メモ】:
- インフォグラフィックを添付
- 元記事のリンクを追加
```

## 5. X(Twitter)スレッド作成プロンプト

### 基本プロンプト

```
以下のNote記事を、X(Twitter)向けのスレッド(3-4ポスト)に分解してください。

【元記事】:
[Note記事の本文を貼り付け]

【スレッドの条件】:
- **ポスト数**: 3-4個
- **各ポストの文字数**: 200-280文字
- **トーン**: カジュアルで親しみやすい
- **絵文字**: 各ポストに2-3個
- **ハッシュタグ**: 各ポストに1-2個

【出力形式】:
以下の形式で出力してください:

---
【ポスト1】(導入)
[本文]

#ハッシュタグ1 #ハッシュタグ2

---
【ポスト2】(ポイント1)
[本文]

#ハッシュタグ1

---
【ポスト3】(ポイント2)
[本文]

#ハッシュタグ2

---
【ポスト4】(まとめ+リンク)
[本文]

📝 詳しくはNote記事で: [URL]

#ハッシュタグ1 #ハッシュタグ2
---
```

## 6. インフォグラフィック作成プロンプト

### Genspark用プロンプト

```
以下の内容で、インフォグラフィックを作成してください。

【記事タイトル】: [タイトル]
【メインメッセージ】: [伝えたい核心メッセージ]

【含める要素】:
1. [要素1]
2. [要素2]
3. [要素3]

【デザイン指定】:
- **サイズ**: 1200x630px(SNS最適サイズ)
- **カラーパレット**: 
  - メインカラー: #[カラーコード]
  - アクセントカラー: #[カラーコード]
  - 背景: #[カラーコード]
- **フォント**: モダンで読みやすいサンセリフ体
- **スタイル**: ミニマル、プロフェッショナル
- **キャラクター**: [統一キャラクターがあれば指定]

【テキスト】:
- タイトル: [タイトル]
- サブタイトル: [サブタイトル]
- 本文: [簡潔な説明]
```

### デザイン統一のための標準設定

```
【標準カラーパレット】:
- メインカラー: #3B82F6 (青)
- アクセントカラー: #F59E0B (オレンジ)
- 背景: #F8FAFC (ライトグレー)
- テキスト: #1E293B (ダークグレー)

【標準フォント】:
- 見出し: 太字、サイズ大
- 本文: レギュラー、サイズ中
- 注釈: ライト、サイズ小

【標準レイアウト】:
- 左上: ロゴまたはアイコン
- 中央: メインメッセージ
- 下部: 補足情報
```

## 7. タグ・ハッシュタグ生成プロンプト

### 基本プロンプト

```
以下の記事に最適なタグとハッシュタグを生成してください。

【記事タイトル】: [タイトル]
【記事概要】: [記事の要約]

【出力形式】:

### Noteタグ(10個)
1. [タグ1]
2. [タグ2]
...

### LinkedInハッシュタグ(5個)
#ハッシュタグ1 #ハッシュタグ2 ...

### Xハッシュタグ(5個)
#ハッシュタグ1 #ハッシュタグ2 ...

【タグ選定の基準】:
- 検索されやすいキーワード
- トレンドに関連するタグ
- ニッチだが関連性の高いタグ
```

## 🎯 使い方のコツ

### 1. プロンプトのカスタマイズ
- [ ]の部分を自分の情報に置き換える
- 必要に応じて条件を追加・削除

### 2. 出力結果の活用
- 生成された内容をそのままGitHubに保存
- テンプレートファイルに沿って整形

### 3. 継続的な改善
- 効果的だったプロンプトは記録
- 改善点があれば随時更新
