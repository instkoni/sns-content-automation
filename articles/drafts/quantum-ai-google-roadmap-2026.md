## 1. 量子AIとは？概念と背景

- **量子アルゴリズム**: Grover検索や量子近似最適化（QAOA）など、特定の最適化問題で指数的高速化が期待される。
- **AIへの応用**: 大規模パラメータ空間のサンプリングや勾配計算を量子回路で高速化し、**量子機械学習（Quantum ML）**と呼ばれる領域が急速に拡大。

> **ポイント**: 現在のAIはGPUベースの数値計算に依存していますが、量子コンピュータは「状態空間の探索」を根本的に変える可能性があります。

## 5. 今すぐ始めるためのステップとツール
1. **Google Cloud アカウント作成** → Quantum AI ベータにアクセス申請。
2. **Python 環境構築** → `pip install tensorflow-quantum` と `pip install qiskit`（Google の SDK と併用可）。
3. **サンプルコード実行**:
```python
import tensorflow as tf
import tensorflow_quantum as tfq
# 量子回路でプロンプト分布をサンプリング
circuit = tfq.convert_to_tensor([tfq.util.create_random_circuit(5, 3)])
sampler = tfq.layers.Sample()
samples = sampler(circuit, repetitions=100)
print(samples)
```
