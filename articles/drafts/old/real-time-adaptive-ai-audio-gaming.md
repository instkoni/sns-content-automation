## 1. リアルタイム適応AIオーディオとは？
- **マルチモーダル入力**: プレイヤーの位置、ヘルス、環境音、ゲームロジックをリアルタイムで取得。
- **AI生成エンジン**: テキスト・音楽・サウンドエフェクトを即座に生成・変形し、シーンに合わせてミックス。
- **低レイテンシ**: 50ms以下の応答で、音楽が瞬時に変化し、没入感が向上。

> **ポイント**: 従来のループ音楽と違い、AIが「シーンを読んで」音楽を「作り変える」ため、プレイヤーの行動と音楽がシンクロします。

## 2. 主な技術スタックと提供サービス
| カテゴリ | 代表サービス | 主な機能 |
|----------|--------------|----------|
| **AI音楽生成** | **Suno AI** | テキストプロンプトから数秒で音楽トラック生成 |
| **リアルタイム音声合成** | **Google Gemini Audio** | 低レイテンシで音声・音楽をストリーミング生成 |
| **ゲームエンジン連携** | **Unity Plugin for Adaptive Audio** | Unity ↔ AIエンジンの双方向通信 |
| **サウンドデザイン** | **AudioCraft** | AIがサウンドエフェクトを即時生成・加工 |

すべて **オンデバイス**（Apple Silicon, RTX GPU）で動作可能で、プライバシー保護とレイテンシ削減が実現します。

## 3. 実践フロー：ゲームに組み込む3ステップ
### 🟢 Step 1：環境データ取得とAPI設定
1. Unity/Unreal のスクリプトで **プレイヤー位置・ステート** を取得し、JSON 形式でエンジンに送信。
2. **Suno AI** の API キーを取得し、`/adaptive-audio` エンドポイントを設定。
3. **Gemini Audio** のローカルモデルをデバイスにデプロイ（GPU最適化）。

### 🟢 Step 2：プロンプト生成と音楽合成
```json
{
  "scene": "boss_fight",
  "mood": "tense",
  "tempo": "140",
  "player_health": 35,
  "environment": "dark_cave"
}
```
- 上記情報を **Suno AI** に送信し、数秒で **BGM トラック** を取得。
- **AudioCraft** で環境音（滴る水音、エコー）をリアルタイム生成し、BGM にミックス。

### 🟢 Step 3：ゲームエンジンへストリーミング
- Unity の **AudioSource** に生成されたバイナリストリームをリアルタイムで注入。
- プレイヤーが **ヘルスが回復** したら、`mood` を `calm` に変更し、再度プロンプトを送信 → 音楽が緩やかに変化。
- **レイテンシ測定**: 50ms 未満で音楽が切り替わることを確認。

## 4. 具体的ユースケースと効果測定
### 🟢 ユースケース1：ステルスミッションの緊張感向上

### 🟢 ユースケース2：レースゲームのダイナミックサウンドトラック
- 車速とコーナー角度を入力し、BGM の **テンポ** と **エレクトロニック要素** を自動調整。
