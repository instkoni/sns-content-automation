# Gemini Title Generator

Gemini 3 Proを使用して、AIO/SEO対策済みのキャッチーなタイトルを3案生成します。

## プロンプト構成要件

以下の要素を含むプロンプトを構築してください。

### 1. AIO（AI Overviews）対策
AIが要約しやすい明確な構造を持つタイトルを生成します。主題と結論が明確で、具体的なキーワードを含むことが重要です。

### 2. SEO対策
検索意図に合致したキーワードを配置します。ターゲットキーワードをタイトル前半に配置し、検索ボリュームのあるキーワードを使用します。

### 3. クリック誘発要素
以下の要素を含めます。

- **数字の活用**: 「3つの方法」「5分で理解」「10のポイント」
- **感情を刺激する言葉**: 「驚愕」「必見」「知らないと損」「衝撃」「話題」
- **具体性**: 「2026年最新版」「実体験レポート」「徹底解説」

### 4. 文字数制限
20〜40文字（日本語）を目安とします。長すぎる場合は画像合成時にフォントサイズが縮小されます。

### 5. ジャンル別トーン

- **人生**: 温かく、インスピレーションを与える、共感を呼ぶ
- **副業**: 実用的、具体的な数字、再現性を強調
- **AI**: 最新性、技術的、未来志向
- **有料**: プレミアム感、限定性、高品質を強調
- **雑記**: カジュアル、親しみやすい、日常的
- **資格**: 信頼性、実績、合格への道筋
- **本業**: プロフェッショナル、キャリア志向、実践的

## プロンプトテンプレート

Gemini 3 Proに以下のプロンプトを送信してください。

あなたはSEO/AIO対策のプロフェッショナルです。
以下の記事ネタに基づいて、Noteプラットフォームで最大限のエンゲージメントを獲得できるタイトルを生成してください。

【記事ネタ】
{topic}

【ジャンル】
{genre}

【ジャンル別トーン】
{genre_tone}

【要件】

1.
AIO（AI Overviews）対策：AIが要約しやすい明確な構造

2.
SEO対策：検索意図に合致したキーワード配置

3.
クリックを誘発する要素：

•
数字の活用（例：「3つの方法」「5分で理解」）

•
感情を刺激する言葉（例：「驚愕」「必見」「知らないと損」）

•
具体性（例：「2026年最新版」「実体験レポート」）



4.
文字数：20〜40文字

6.
強調表示
もっとも重要なキーワード（1つまたは2つ）を `*`（アスタリスク）で囲んでください。これはサムネイル画像で赤文字として強調されます。

【出力形式】
タイトル案を3つ提案してください。番号付きリストで出力してください。

1.
[タイトル案1（例：*ChatGPT*で時短する3つの*神技*）]

2.
[タイトル案2]

3.
[タイトル案3]

## 使用方法

エージェントは、このSkillを自動的に検出し、Gemini 3 Proにプロンプトを送信します。生成された3つのタイトル案をユーザーに提示し、選択を促してください。
