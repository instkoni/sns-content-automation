# Note記事サムネイル自動生成システム

このプロジェクトは、Google Antigravity IDEで動作するAIエージェントのためのAgent Skillsを提供し、Note記事のサムネイル作成を自動化します。

## 概要

3つのAgent Skillsが連携し、以下の処理を自動化します。

1.  **タイトル生成**: Gemini 3 ProによるAIO/SEO対策済みタイトルの生成
2.  **イラスト生成**: Nano Banana Proによる高品質なイラストの生成
3.  **画像合成**: Canvaテンプレート、イラスト、タイトルを合成
4.  **ファイル保存**: 指定パスへの自動命名による保存

## セットアップ

1.  **Antigravityで開く**: このプロジェクトをAntigravity IDEでワークスペースとして開きます。
2.  **リソースの配置**:
    - `templates/`ディレクトリに、ジャンル名のPNGファイル（例: `人生.png`）を配置します。
    - `fonts/`ディレクトリに、日本語フォントファイル（例: `NotoSansJP-Bold.ttf`）を配置します。

## エージェントへの指示例

Antigravity IDEのエージェントに、以下のように指示してください。

Note記事のサムネイルを作成してください。

ジャンル: [ジャンル名]
記事ネタ: [テーマやキーワード]

エージェントは自動的にSkillを検出し、対話形式で処理を進めます。
