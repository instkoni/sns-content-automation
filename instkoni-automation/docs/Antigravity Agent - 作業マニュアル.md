# Antigravity Agent - 作業マニュアル

## 概要

Antigravityのエージェントチャットで各種タスクを実行するための、指示コマンドをまとめたマニュアルです。このマニュアルを参照することで、作業をスムーズに進めることができます。

---

## 1. 15本のニュース記事を自動生成する（N8N連携）

### 目的
N8Nワークフローで取得した最新ニュースのタイトル案に基づき、15本のNote記事を全自動で生成します。

### 指示コマンド
```
note-article-generatorスキルを使って、15本のNote記事を生成してください
```

### 実行フロー
1. N8Nワークフローを実行し、Antigravityを起動します。
2. エージェントチャットに上記の指示コマンドをコピー＆ペーストします。
3. エージェントが自動的に`/Volumes/WDBLACK_2TB/Git/sns-content-automation/pipeline_status.md`を読み込み、記事生成を開始します。

### 補足
- このタスクは、N8Nによって`pipeline_status.md`が正しく作成されていることが前提です。

---

## 2. 対話形式でテーマファイルを作成する

### 目的
新しいテーマで記事を作成する際に、エージェントと対話しながらテーマファイル（`current_theme.md`）を作成します。

### 指示コマンド
```
theme-generatorスキルを使って、カスタムテーマファイルを作成してください
```

### 実行フロー
1. Antigravityを起動します。
2. エージェントチャットに上記の指示コマンドをコピー＆ペーストします。
3. エージェントからの質問（メインテーマ、読者層、記事数など）に答えます。
4. エージェントがウェブ検索で最新情報を調査し、タイトル案を提案します。
5. 提案されたタイトル案にフィードバックし、内容を確定させます。
6. 最終的に、`/Volumes/WDBLACK_2TB/Git/sns-content-automation/custom_themes/current_theme.md`が自動で作成・保存されます。

### 補足
- 新しいテーマで記事を書きたい場合に、最初に実行するタスクです。

---

## 3. 個別のテーマで記事を作成する

### 目的
`current_theme.md`に定義されたテーマとタイトル案に基づき、複数の記事を自動生成します。

### 指示コマンド
```
custom-theme-articleスキルを使って記事を生成してください
```

### 実行フロー
1. 上記の「2. 対話形式でテーマファイルを作成する」を実行し、`current_theme.md`を準備します。
2. エージェントチャットに上記の指示コマンドをコピー＆ペーストします。
3. エージェントが自動的に`current_theme.md`を読み込み、記事生成を開始します。

### 補足
- このタスクは、`current_theme.md`が事前に準備されていることが前提です。

---

## ワークフロー全体像

**N8N連携（定型タスク）**
```
N8N実行 → Antigravity起動 → 指示コマンド1 → 15本の記事生成
```

**個別テーマ（非定型タスク）**
```
Antigravity起動 → 指示コマンド2 → 対話 → current_theme.md作成 → 指示コマンド3 → 複数記事生成
```
